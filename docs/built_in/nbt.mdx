# NBT

Substitutes the NBT-value at a given path as retrieved from various sources; this tag acts as
a placeholder and must thereby be self-closing (does not accept content).

## Block

### Overview

Retrieves data from a block within the player's world.

<TagDocRender tag={{
  name: 'block-nbt',
  aliases: [],
  selfClosing: true,
  attributes: [
    {
      name: 'coordinates',
      aliases: [],
      description: 'Coordinates of the target block',
      fallback: null,
      multiValue: false,
      type: AttributeType.COORDINATES,
    },
    {
      name: 'path',
      aliases: [],
      description: 'Path of the NBT-value',
      fallback: null,
      multiValue: false,
      type: AttributeType.STRING,
    },
    {
      name: 'separator',
      aliases: [],
      description: 'Separator injected in-between list-items',
      fallback: 'null',
      multiValue: false,
      type: AttributeType.MARKUP,
    },
    {
      name: 'representation',
      aliases: [],
      description: 'Representation of each item; introduces the {value} placeholder',
      fallback: 'null',
      multiValue: false,
      type: AttributeType.MARKUP,
    },
  ]
}} />
