# Word-Wrap

## Overview

Wraps text at a given maximum line length, breaking at spaces where possible and allowing to
inject custom separators in-between values, if there are multiple of them.

## Examples

### Single Value

```!component-markup
<&6><word-wrap
  value="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
  width=32
/>
```

### Multiple Values

```!component-markup
<&6><word-wrap
  [@value]="['Pink Terracotta', 'Red Sand', 'Diamond Ore', 'Golden Leggings', 'Deepslate Emerald Ore', 'Raw Salmon']"
  value-separator={<&7>,<space/>}
  width=25
/>
```

In order to attach a custom appearance to only the value-tokens, but not the value-separators,
make use of the `token-renderer` attribute.

```!component-markup
<&6><word-wrap
  [@value]="['Pink Terracotta', 'Red Sand', 'Diamond Ore', 'Golden Leggings', 'Deepslate Emerald Ore', 'Raw Salmon']"
  value-separator={<&7>,<space/>}
  token-renderer={<u>{token}}
  width=25
/>
```

### Value Shorthand

For convenience, the `value` attribute also supports a bound expression of arbitrary name,
while all other attributes support bound expressions with matching names.

```!component-markup
<container
  *let-my_value="['Pink Terracotta', 'Red Sand', 'Diamond Ore', 'Golden Leggings', 'Deepslate Emerald Ore', 'Raw Salmon']"
  *let-value_separator={<&7>,<space/>}
>
  <&6><word-wrap
    [@my_value]
    [value_separator]
    token-renderer={<u>{token}}
    width=25
/>
```
