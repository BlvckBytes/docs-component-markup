## Examples

### Fixed Decimals

Formatting a fractional value of arbitrary complexity to a visually appealing two-decimals number.

```!component-markup
This item costs <&b><number value=512.4582831 format="0.00"/> Coins
```

### Absolute Values

Removing the negative-sign `-`, if any.

```!component-markup
My absolute value: <&b><number value=-212 abs/>
```

### Whole Numbers

Dropping all decimal digits, if any.

```!component-markup
My whole number: <&b><number value=3.1415926 int/>
```

### Padding

Adding additional leading zeroes, as to ensure a fixed width.

```!component-markup
<container *for-place="7..11" *for-separator={<br/>}>
  <&7>#<number [value]="place" format="00"/></> - <&b>Name
```
