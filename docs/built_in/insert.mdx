# Insert

## Overview

Inserts text into the player's chat-prompt at the position of the cursor when shift-clicking.

<TagDocRender tag={{
  name: 'insert',
  aliases: [],
  selfClosing: false,
  attributes: [
    {
      name: 'value',
      aliases: [],
      description: 'Text to insert when shift-clicking',
      fallback: null,
      multiValue: false,
      type: AttributeType.STRING,
    }
  ]
}} />

## Example

### Simple Insertion

```!component-markup
<&7>Click me to insert:<space/>
<insert value="Example"><&d>Example</insert>
```

### Value Shorthand

For convenience, the `value` attribute also supports a bound expression.

```!component-markup
<container +let-my_value="Example">
  <&7>Click me to insert:<space/>
  <insert [my_value]><&d>{my_value}</insert>
```
