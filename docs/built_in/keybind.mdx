# Keybind

## Overview

Substitutes the configured keybind corresponding to the given action; this tag acts as
a placeholder and must thereby be self-closing (does not accept content).

<TagDocRender tag={{
  name: 'key',
  aliases: [],
  selfClosing: true,
  attributes: [
    {
      name: 'key',
      aliases: [],
      description: 'Path of the key to render',
      fallback: null,
      multiValue: false,
      type: AttributeType.STRING,
    }
  ]
}} />

## Example

```!component-markup
Press <u><&a><key key="key.jump"/></u> to jump! :)
```

```!component-markup
Hold down <u><&b><key key="key.forward"/></u> to move ahead.
```
