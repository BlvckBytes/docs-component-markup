# Duration

## Overview

## Examples

### Default Style

When just supplying a value as well as some units to format into, the default style will be plain but efficient:

```!component-markup
My duration: <&b><duration
  [value]="(31 + 60*3 + 60*60*8) * 1000"
  units="hms"
/>
```

### Custom Style

As to really show what's possible, let's not just render different unit-suffixes, but also account for singular/plural differentiation, color groups based on their unit and add a colored separator.

```!component-markup
My duration: <duration
  [value]="(38 + 60 + 60*60*2) * 1000"
  units="smh"
  separator={<&8>,<space/>}
  unit-renderer={
    <container *when="unit">
      <&c +is="h">{value} hour{value != 1 ? 's'}</>
      <&a +is="m">{value} minute{value != 1 ? 's'}</>
      <&b +is="s">{value} second{value != 1 ? 's'}</>
  }
/>
```
